HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • In the head section you have information such as: Encoding type, Meta data, Title & viewport.
  • The Body section contains information such as: Links, Headers, Nav bars, sections, footers & images.

CSS

  • CSS predominantly element names as selectors to style the HTML.
  • There are 3 ways to style the HTML elements: Element name, ID & class.
  • You can define multipule elements style by typing the element names seperated by a comma ,.
  • When using a class to select an element to set the style it can be more than one type of element.
  • when setting the style for an element you simply type the name, when selecting the element by the class you start with a period followed by the class name and when selecting an element by the ID you start with a pound (hash) sign followed by the ID name.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: ads all the files from the current project ready to commit.
  • git commit -m "message here": stages the files ready to send to git.
  • git push origin "directory here": pushes the staged files to git repository.
